# Data Access and Compliance

This repository provides code only.

## CFPS Data

- CFPS data are subject to their own access and usage terms.
- Users must obtain data through official channels and comply with all license/ethics requirements.
- Do not upload raw CFPS files or redistributed extracts to this repository.

## Sensitive Information

- Keep personal path configuration in local `config.do` only.
- Never commit private credentials or personal identifiers.
